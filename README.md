## Hospital Emergency Appointment Registration System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6528fa16bc13446481c045a647be39c6/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, doctors, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Ward Information
5. Forum Information
6. Announcement Information
7. Doctor Information
8. Personal Center
9. Ward Booking
10. Emergency Booking


**Doctor Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Forum Information
6. Announcement Information
7. Emergency Booking
8. Medical Record Information



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Ward Information Management
5. Ward Booking Management
6. Medical Record Information Management
7. Basic Data Management (Ward Type Management, Announcement Type Management, Department Management, Time Type Management, Position Management)
8. Forum Information Management
9. Announcement Information Management
10. Doctor Management
11. Emergency Booking Management
12. User Management
13. User Health Code Management
14. Carousel Information Management
